# github-actions-cicd-example docs hosting with GitHub pages

Docs hosting for the
[alexmalins/github-actions-cicd-example](https://github.com/alexmalins/github-actions-cicd-example)
project using GitHub Pages.

Changes to the docs source on that repo are automatically propagated into the
HTML files in this repo by a GitHub Action. This means the docs are always up
to date at
[https://alexmalins.github.io/github-actions-cicd-example/](https://alexmalins.github.io/github-actions-cicd-example).
